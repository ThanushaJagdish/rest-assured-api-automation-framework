API Test Automation Framework

Overview

This project is an API automation framework built using Rest Assured, TestNG, and Maven.
It validates banking APIs and also includes data-driven testing (Excel).

Features

* API automation using Rest Assured
* Data-driven testing with TestNG DataProvider
* End-to-end user flow (Login → Transactions → Transfer → Feedback → Logout)
* Request & Response logging
* Extent & Allure reporting
* JSON schema validation

---

How to Run

In bash
mvn clean verify
```
